# todo-cli

Simple todo list tracker using cli

<p align="center">
    <img src="https://img.shields.io/badge/golang-v1.23-blue" height="25"/>

# Features

- create
- print (list or board)
- update status (todo, in progress, done)
- delete

How to use:

```bash
go run main.go
```
